Open Source

Babl is open-source and available on GitHub: https://github.com/babl-ws/babl.

Babl is built using these wonderful open-source projects:

  • Agrona - Fast, cache-friendly, allocation-free collections
  • Aeron - Low-latency message transport
  • JMH - Benchmarking framework
