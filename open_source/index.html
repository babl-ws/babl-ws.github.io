Open Source

Babl is open-source; the source code is available on GitHub: https://github.com/babl-ws/babl.

Babl is built using several other wonderful open-source products:

  • Agrona - Fast, cache-friendly, allocation-free collections
  • Aeron - Low-latency message transport
  • JMH - Benchmarking framework
